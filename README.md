# NetCoreDemoApi
API made with dotnet core 6, with didactic purpose.

# Installation
```
git clone https://github.com/vhspicerosGitHub/net-core-demo-api.git
```

# Compile
```
dotnet build
```

# Run Tests
```
dotnet test
```

# Usage
```
dotnet run --project  .\NetCoreDemoApi.Web\NetCoreDemoApi.Web.csproj
```
